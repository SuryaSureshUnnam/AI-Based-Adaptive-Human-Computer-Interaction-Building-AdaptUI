How adaptation works:

AdaptUI continuously monitors user interactions — clicks, chat usage, scroll depth, and tab switching. A lightweight behaviour classifier runs client-side and updates your profile (Novice → Explorer → Power User → Expert) in real time. When a threshold is reached, the interface applies adaptive changes such as toggling dark mode, surfacing advanced features, or restructuring the layout.

About AdaptUI

AdaptUI is a research prototype built for MSAI-631 – AI for Human-Computer Interaction at the University of the Cumberlands (Spring 2026). It demonstrates AI-based adaptive UI principles by tracking real-time behavioural signals and dynamically personalising the interface without any manual configuration by the user.


Key concepts implemented: Adaptive User Interfaces (AUI), Intelligent User Interfaces (IUI), and AI-based Adaptive HCI. The AI assistant uses the Anthropic Claude API to provide context-aware responses that also adjust based on the detected user profile.
